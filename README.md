This project is a recipe-centric menu building and optimization program used for personal development. The main focus of this project is to utilize the scrum method as well as work with various design patterns in a real-world application.

##Features##
- **comprehensive recipe storage**: including information such as ingredient, time, serving size, macronutrient, and allergy risks
- **advanced filtering and searching**
- **automated and manual menu generation**
- **price per serving estimation**
                    *NOTE: currently in development, prices only applicable to California, USA*

##Technology##
- **Language**: Java
- **Database backend**: SQLite
- **Project management**: JIRA (scrum)
- **version control**: Github
